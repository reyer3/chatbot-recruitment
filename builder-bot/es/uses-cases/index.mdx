## My first bot

El siguiente c√≥digo representa el uso r√°pido de un bot que cuando escribes la palabra `hola`, te saluda con un mensaje de bienvenida y te pregunta tu nombre y luego devuelve una imagen divertida

        const [imageUrl] = await dataApi.json()
        await flowDynamic([{body:'üòú', media: imageUrl}])
    })

    const adapterDB = new MemoryDB()
    const adapterFlow = createFlow([welcomeFlow])
    const adapterProvider = createProvider(BaileysProvider)

    adapterProvider.initHttpServer(3000)

    await createBot({
        flow: adapterFlow,
        provider: adapterProvider,
        database: adapterDB,
    })

## Turn off bot a certain user {{ tag: 'state'  }}

A veces necesitaremos apagar el barco para un determinado usuario, para que podamos tener una conversaci√≥n con el cliente sin que el bot interfiera.

```ts
const flow = addKeyword<BaileysProvider>('magic keyword')
  .addAction(async (_, { state, endFlow }) => {
      const botOffForThisUser = state.get<boolean>('botOffForThisUser')
      if(botOffForThisUser) return endFlow()
  })
  .addAnswer('Hello!')
```

## Turn off for everyone {{ tag: 'state'  }}

  })
  .addAnswer('Hello!')
